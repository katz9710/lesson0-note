HTML stands for Hyper-Text-Markup-Language. HTML is responsible for most of what information will be displayed and how it displayed on user's computer. text-content describe what user sees and markup describes how it looks. Tag, Markup and element

HTML is made up tags which start with bracket and end with slash in bracket. This tag tells how the text between these tags are formatted. Markup is how the tag is maeking the text to show the style of these texts. Taken whole tags and texts are called element and stacking up these elements makes web page.
